# LootGen

Утилита для симуляции выпадения лута из сундуков.

## Технологии
- Python
- rich (красивые таблицы)
- ruff (линтер)
- GitHub Actions (CI)

## Как запустить (одной командой):

1. `git clone https://github.com/твой_ник/LootGen.git`
2. `cd LootGen`
3. `python -m venv venv`
4. `venv\Scripts\activate` (на Windows) или `source venv/bin/activate` (Mac/Linux)
5. `pip install -r requirements.txt`
6. `python lootgen.py`

Готово! Увидишь красивую таблицу.
